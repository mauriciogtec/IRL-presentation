IRL Problem

We can interpret it as a maximum likelihood problem

$$ \max_\theta ; \mathbb{E}{\tau \in \mathcal{D}}\left[\log p\theta(\tau)\right] $$

where $p_\theta(\tau)=\frac{1}{Z}\exp(-c_\theta(\tau))$ is parametrized by the Boltzmann distribution.

Typical problem with Boltzmann distribution is estimating the partition function $Z$. Suppose we can sample from another policy $q$ different from the true data distribution $p$. Then $$ Z= \int \exp(-c_\theta(\tau)) d\tau=\int \exp(-c_\theta(\tau)\frac{q(\tau)}{q(\tau)} d\tau = \mathbb{E}{\tau \sim q} \left[ \frac{\exp(-c\theta(\tau))}{q(\tau)} \right] $$ Then we have a loss function $$ \begin{aligned} l(\theta) & = \mathbb{E}{\tau\sim p}[-\log p\theta(\tau)] \ & = \mathbb{E}{\tau \sim p}[c\theta(\tau)] + \log Z \ & = \mathbb{E}{\tau \sim p}[c\theta(\tau)] + \log \mathbb{E}{\tau \sim q} \left[ \frac{\exp(-c\theta(\tau))}{q(\tau)} \right] \end{aligned} $$
